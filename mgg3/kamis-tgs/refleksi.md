# Refleksi

 1. Mengapa closure sering digunakan untuk menyimpan data private?

Closure memungkinkan data tetap tersimpan di dalam function dan tidak dapat diakses langsung dari luar. Hal ini membuat data lebih aman dan terkontrol.

 2. Apa keuntungan menggunakan scope yang baik dalam program besar?

Scope membantu mengatur variabel agar tidak saling bertabrakan, membuat kode lebih rapi, mudah dibaca, dan lebih mudah diperbaiki.

 3. Apa risiko penggunaan hoisting yang tidak dipahami dengan baik?

Hoisting dapat menyebabkan error atau hasil yang tidak sesuai harapan karena variabel atau function digunakan sebelum deklarasi.

 4. Mengapa Date Object penting dalam aplikasi nyata?

Date Object digunakan untuk mengelola tanggal dan waktu, seperti sistem login, jadwal kegiatan, absensi, pengingat acara, dan perhitungan umur.
