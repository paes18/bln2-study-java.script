# Ringkasan JavaScript Dasar     

## Bab 2 - Syntax, Statement, dan Comment

* Syntax adalah aturan penulisan kode.
* Statement adalah perintah yang dijalankan program.
* Comment adalah catatan yang tidak dijalankan program.

---

## Bab 3 - Variable

* Digunakan untuk menyimpan data.
* Contoh:

```js
let nama = "faiz";
```

---

## Bab 4 - Tipe Data

* String → teks
* Number → angka
* Boolean → true atau false
* Array → banyak data dalam satu variable
* Object → data yang memiliki beberapa informasi

---

## Bab 5 - Operator

* Aritmatika → `+ - * / %`
* Perbandingan → `> < >= <= == ===`
* Logika → `&& || !`

---

## Bab 6 - Percabangan

* Digunakan untuk mengambil keputusan.
* Menggunakan:

  * `if`
  * `if else`
  * `else if`

---

## Bab 7 - Perulangan

* Digunakan untuk mengulang proses.
* Menggunakan:

```js
for
```

---

## Bab 8 - Array dan Array Method

* Array digunakan untuk menyimpan banyak data.
* Method yang dipelajari:

  * `push()`
  * `pop()`
  * `shift()`
  * `unshift()`
  * `includes()`
  * `indexOf()`

---

## Bab 9 - Function

* Kumpulan kode yang bisa digunakan berulang kali.
* Membuat program lebih rapi dan mengurangi kode yang berulang.

---

## Bab 10 - Object dan Method

* Object digunakan untuk mengelompokkan data.
* Property = data di dalam object.
* Method = function di dalam object.

---

## Bab 11 - Iterable dan Non-Iterable

### Iterable

* Bisa diulang menggunakan perulangan.
* Contoh:

  * Array
  * String

### Non-Iterable

* Tidak bisa langsung diulang menggunakan `for...of`.
* Contoh:

  * Object

---

# Kesimpulan

JavaScript dasar mencakup cara menyimpan data, mengolah data, mengambil keputusan, melakukan perulangan, membuat function, menggunakan object, serta memahami data yang bisa dan tidak bisa di-loop.
