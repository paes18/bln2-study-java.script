# Refleksi

## 1. Kapan Closure lebih baik digunakan dibanding variabel global?

Closure lebih baik digunakan ketika data tidak boleh diakses atau diubah langsung dari luar program. Dengan closure, data tetap tersimpan tetapi hanya bisa diubah melalui method yang sudah dibuat.

## 2. Mengapa Spread Operator sering digunakan saat mengolah data?

Spread Operator sering digunakan karena bisa menyalin dan menggabungkan array atau object tanpa mengubah data asli. Ini penting supaya data awal tetap aman.

## 3. Apa risiko jika tidak memahami Scope dan Hoisting?

Jika tidak memahami scope, kita bisa salah mengakses variabel dan menyebabkan error. Jika tidak memahami hoisting, kita bisa bingung kenapa variabel bernilai undefined atau kenapa let dan const error saat dipanggil sebelum deklarasi.

## 4. Mengapa Date Object penting dalam aplikasi nyata?

Date Object penting karena banyak aplikasi membutuhkan tanggal dan waktu, seperti transaksi bank, riwayat aktivitas, jadwal acara, sistem parkir, dan laporan transaksi.

## 5. Menurutmu materi mana yang paling sulit? Jelaskan alasannya

Menurut saya materi paling sulit adalah closure, karena kita harus memahami bagaimana function bisa tetap mengingat variabel dari scope luarnya. Awalnya terlihat membingungkan, tetapi setelah dipakai untuk sistem bank atau leaderboard, fungsinya jadi lebih jelas.
